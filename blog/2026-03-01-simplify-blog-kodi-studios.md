# Simplify Blog Workflow for Kodi Studios Website

Kodi Studios is an old website, it has gone through multiple iterations. Some
were more successful than others. Started off with WordPress, then moved to more
modern and sustainable solutions like Docusaurus. It really takes an effort to
get website to good condition. However, if approached one step at a time, this
big effort can be fully achieved.

One lesson is to not get overwhelmed. Tech stack changes every other year, but
as long as you keep and control your content, you have the power to migrate,
transform it.

Don't be afraid to start new website, even in 2026 with social media all around.
Social media basically controls what it shows and what it doesn't. Your own
website is your own presence on internet.

## Accomplishments

### Kodi Studios Website https://kodistudios.com

- 🖼️ Blog \
  Migrated to new blogging pattern that removes noise and allows creator to
  concentrate on blog content and message. \
  One of core improvements is enabling markdown formatting, which improves
  readability not only for rendered blog pages, but also for original source
  markdown files. Super Excited for this improvement. \
  Blogging can become a pleasant zen activity, rather than a battle with
  formatting utilities.
- ⛵ Linters \
  Added Linters and other Best Practices into Kodi Studios website. It should
  make it easier to maintain, update.
